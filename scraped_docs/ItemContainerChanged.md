An event called whenever the stack size of an [`Item`](../Item.html "class in net.runelite.api")
in an [`ItemContainer`](../ItemContainer.html "interface in net.runelite.api") is modified.

Examples of when this event may trigger include:

+ Withdrawing an item from bank (triggers for both bank and player inv)
+ Picking up an item
+ Dropping an item
