Posted every client tick (20ms).

Client ticks are roughly broken down into
1. packet processing
2. interface tick
3. client tick (this event)
4. clientscript execution
5. menu sorting (see [`PostMenuSort`](PostMenuSort.html "class in net.runelite.api.events"))
6. menu click detection and drag handling
7. [`PostClientTick`](PostClientTick.html "class in net.runelite.api.events") event
